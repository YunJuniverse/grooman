# grooman 브랜드기획서

> 작성 기준: `20_guides/14_브랜드기획서_작성_지침.md` §8.1 목차 + §6.14~6.21(2025-26 표준) + §19 craft.
> 상위 입력: `30_planning/10_사업기획서.md` v1(포지셔닝·비목표) — 브랜드는 사업 방향을 *고객 인식의 언어*로 번역한다(지침 §10.2).
> `[가정]` = 검증 전 가정, `[미확정]` = 사람 결정 필요.

## 1. 브랜드 개요

- **브랜드명**: grooman (그루맨) — grooming + man. 도메인 grooman.kr.
- **한 줄 정의**: 대한민국 남성이 그루밍 정보를 찾을 때 **광고가 아닌 실사용 경험**으로 답하는 개방형 커뮤니티.
- **존재 이유**: 남성 그루밍 소비는 급증했지만(사업기획서 §2), 구매 전 정보 단계는 광고 블로그·파편화된 익명 게시판뿐이다. grooman은 "검색하면 닿는 중립적인 남성 그루밍 정보처"가 되기 위해 존재한다.
- **브랜드가 약속하는 것**: ① 특정 제품·병원에 기울지 않는다 ② 출처를 숨기지 않는다 ③ 경험담을 광고보다 위에 둔다.
- **문서 범위**: 인식·메시지·톤·비주얼·경험 원칙의 원본. 시장성·BM은 사업기획서, 기능·정책은 서비스기획서(예정), 채널 실행은 마케팅기획서(예정)가 원본.

## 2. 시장 인식 분석 (§19.1 인식 맵)

| 경쟁/대안 브랜드 | 포지셔닝 | 톤 | 고객 인식 |
|---|---|---|---|
| 대다모 | 탈모 전문 커뮤니티 | 절실·동병상련 | "탈모는 여기" — 단일 주제, 그루밍 전반 아님 |
| 디시 갤러리(그루밍 관련) | 익명 잡담 | 거칠고 냉소적 | 정보는 있으나 노이즈·광고 혼재, 신뢰 유보 |
| 네이버 카페(남성 뷰티류) | 폐쇄형 회원제 | 상업적·등업 문화 | 가입해야 보임, 업체 침투 의심 |
| 화해·글로우픽 | 성분·리뷰 데이터 앱 | 정제·데이터 중심 | 신뢰하나 "여성용"이라는 인식 |
| 유튜브 그루밍 채널 | 인플루언서 개인 브랜드 | 활기·협찬 혼재 | 재밌지만 "뒷광고" 경계심 |

- **고객이 중요하게 보는 기준**: 광고인지 아닌지(신뢰) > 나 같은 사람의 경험인지(공감) > 찾기 쉬운지(접근성).
- **포지셔닝 공백**: **"남성 전용 × 전 영역 × 열려 있고(검색) × 편들지 않는(중립)"** 인식 좌표가 비어 있다 — 어느 대안도 4개를 동시에 점유하지 못함.

## 3. 브랜드 포지셔닝 (Dunford 5요소 도출 — §6.14)

> 도출 순서 준수 (결과 문장보다 순서가 규율):

1. **경쟁 대안** (우리가 없으면 고객이 실제로 하는 행동): 네이버/구글 검색 → 광고성 블로그·지식인 → 반신반의 → 디시·오픈채팅에서 재확인. *phantom competitor 없음 — 위 §2 표의 실제 대안만.*
2. **고유 속성**: ① 남성 그루밍 전 영역 단일 허브 ② 비로그인 열람 + 검색 색인(개방형) ③ 중립성의 구조화 — 시술(clinic) 카테고리 자동수집 금지·UGC 전용, 자동수집 글 출처·뱃지 표기 ④ AI 큐레이션으로 항상 신선한 콘텐츠.
3. **가치 (So what? + 증거)**: "검색 한 번으로, 편들지 않는 실경험 정보에 바로 닿는다" — 증거: 개방형 색인 구조(구현됨), 출처 표기(구현됨), clinic UGC 전용(AI 분류 단계 차단, 구현됨).
4. **핵심 세그먼트**: 그루밍 입문~중급 남성(정보 탐색 단계) — 특히 탈모·스킨케어 첫 고민 시점.
5. **시장 카테고리**: **"남성 그루밍 커뮤니티"** — 기존 카테고리 내 포지셔닝을 택한다(신규 카테고리 창출 아님 → Class C 스코프 약속 없음, 지침 §6.14 분기 결정 명시).

- **포지셔닝 문장**: "그루밍이 궁금한 남성에게, grooman은 광고 없이 실경험으로 답하는 유일한 개방형 종합 커뮤니티다."
- **피해야 할 이미지**: 업체 침투 카페 ✕ · 남성성 강요("진짜 남자라면") ✕ · 뷰티 잡지식 화려함 ✕ · 의료 상담소 ✕.

## 4. 타깃 페르소나

| 항목 | 내용 |
|---|---|
| 핵심 타깃 | 25~40세 남성, 그루밍 고민이 *처음 생긴* 시점(탈모 초기 징후·피부 트러블·첫 향수) |
| 문제/욕구 | 물어볼 곳이 없다 — 친구에겐 민망하고, 검색 결과는 광고 같고, 여성 커뮤니티는 내 맥락이 아니다 |
| 중요하게 보는 가치 | "이거 광고 아니야?"를 통과하는 신뢰 · 나와 같은 단계의 경험담 · 빠른 답 |
| 선택 이유 | 검색에서 바로 열리고(가입 강요 없음), 출처가 투명하고, 같은 고민의 남성들이 있다 |
| 기대 정서 | 안심(혼자가 아님) · 존중(무시당하지 않음) · 실용(바로 써먹는 정보) |

## 5. 브랜드 핵심 가치 (§19.2 — 가치마다 실체 강제)

우선순위 순. **"증명하라, 선언하지 마라"** — 각 가치의 실체는 구조·코드·정책이다.

| # | 가치 | 의미 | **이 가치의 실체** (반박 불가능한 구조) | 금지되는 반대 행동 |
|---|---|---|---|---|
| 1 | **중립** | 특정 제품·병원 편을 들지 않는다 | 시술(clinic) 카테고리 자동수집 금지 — AI 분류 enum에서 구조적 차단(구현됨) · 스팸/광고 필터(spam_score·is_advertisement) | 협찬 미표기 콘텐츠, 특정 병원 추천 배너 |
| 2 | **투명** | 콘텐츠의 출처와 성격을 숨기지 않는다 | 자동수집 글 "자동수집" 뱃지 + 원문 출처 링크(구현됨) · 테스트 봇은 배포 전 전량 삭제(ADR-0002 릴리스 게이트) | 자동 글의 사람 위장, 뒷광고 |
| 3 | **실경험 우선** | 광고 문구보다 사용자의 경험담이 위 | UGC 후기 구조(레벨·추천·HOT보드) — 경험담이 상단에 오르는 랭킹 | 브랜드 보도자료 상단 고정 |
| 4 | **남성의 맥락** | 남성의 언어·고민 단계로 말한다 | 6개 카테고리 설계 자체(hair/skin/shaving/fragrance/clinic/deals) — 남성 탐색 동선 기준 | 여성 뷰티 문법 이식, 남성성 훈계 |

## 6. 브랜드 아키텍처 · 버벌 아이덴티티 (§6.16)

- **아키텍처**: **Branded House 단일 브랜드** — grooman 하나로 모든 카테고리를 커버(salience 집중). 서브브랜드 없음. 카테고리명(hair/skin/…)은 브랜드가 아니라 *메뉴* — 별도 lockup·명명 금지.
- **네이밍 규칙(nomenclature)**: 기능·게시판은 한국어 보통명사("핫딜", "시술·성형 후기") — 조어·영문 신조어 만들지 않음. 표기: 로고·헤더 = `GROOMAN`(대문자), 본문 = "그루맨".
- **Tagline(항구적) 후보** [미확정 — 사람 선택]: ① "남자의 그루밍, 경험으로 답하다" ② "광고 말고, 경험" ③ "그루밍이 궁금한 남자들의 커뮤니티". *slogan(캠페인 한시)은 마케팅기획서 소관.*
- **금칙어 lexicon** (§9.7·의료법·표시광고 리스크 반영):
  - 효능 단정: "완치", "100% 효과", "부작용 없음", "무조건"
  - 의료적 단정: "진단", "처방" 뉘앙스의 조언 문구(정보 제공으로 한정)
  - 남성성 강요: "남자라면", "남자답게", 외모 비하·조롱 표현
  - 과장 수식: "국내 최고", "유일무이"(근거 없이), "혁명적"

## 7. 브랜드 메시지 체계 (§19.3 다면 메시지 하우스)

| 위계 | 정보탐색자 (검색 유입) | 기여자 (UGC 작성자) | 파트너 (P3 브랜드·광고주) |
|---|---|---|---|
| 태그라인 | (§6 확정 후 공통) | ← | ← |
| 핵심 메시지 | "광고 말고 진짜 경험을 찾는다면" | "당신의 실패담이 누군가의 지름길" | "신뢰를 쌓은 커뮤니티에서, 광고임을 숨기지 않고" |
| 보조 메시지 | "가입 없이 바로 보세요" / "출처를 항상 밝힙니다" | "같은 고민, 같은 단계의 남자들" / "레벨은 경험의 증명" | "중립성은 팔지 않습니다 — 지면은 팝니다" |
| 금지 메시지 | "최저가", "추천 1위" 류 커머스 화법 | "등업하려면" 류 폐쇄 카페 화법 | 네이티브 광고 위장 제안 |

- 적용 예시(검색 랜딩 첫 문장): "미녹시딜 6개월 써본 사람들의 기록입니다 — 광고가 아니라."

## 8. 톤앤매너 (§19.4 정량화 + §6.19 voiceProfile)

**voiceProfile 4축** (1~5):

| 축 | 값 | 해석 |
|---|---|---|
| 격식 | 2 | 존댓말이되 캐주얼 — "~해요"체 기본, 공지·정책만 "~합니다"체 |
| 에너지 | 3 | 차분하지만 무겁지 않게 — 핫딜(deals)만 4 허용 |
| 따뜻함 | 4 | 공감 우선 — 고민 글엔 판단보다 경험 공유 |
| 복잡성 | 2 | 입문자 기준 — 전문용어는 첫 등장 시 한 줄 풀이 |

**정량화 톤 규칙 — "모호 수식어 금지, 수치 우선"** (변환 예시 쌍):

| 금지 (모호) | 권장 (구체) |
|---|---|
| "효과 좋은 샴푸" | "3개월 사용 후 세면대 빠짐 개수 기준 절반" |
| "빠른 배송" | "결제 다음날 도착" |
| "많은 남성들이" | "이 글 추천 128개" |

**예시 문장** (봇 페르소나 아카이브에서 추출한 보이스 — 페르소나는 삭제 예정이나 문체 자산은 승계):
- ✅ "M자 3년차입니다. 미녹시딜부터 이식 상담까지 다 해봤고, 실패한 것부터 말씀드릴게요."
- ❌ "혁신적인 탈모 솔루션을 소개합니다!" (금칙: 과장·광고 화법)

**AI 어시스턴트 페르소나 (§6.11)**: 해당 없음 — 사용자향 챗봇 없음. AI는 백엔드 분류·요약에만 사용(자동수집 뱃지로 표기). 도입 시 본 절 신설 필수.

## 9. 비주얼 방향 (+ DBA §6.15 · WCAG §6.20 · 모션 §6.21)

**현재 구현 상태 (실측)**: 워드마크 `GROOMAN` 텍스트 로고 · Pretendard 서체 · 액센트 인디고 `#4f46e5` · 라이트/다크 듀얼 테마(다크: `#10131a` 계열) · fade-in/slide-up 마이크로 모션.

- **무드**: "정돈된 정보 밀도" — 퀘이사존의 정보성 + 화해의 정제됨, 뷰티 잡지의 화려함 ✕. 콘텐츠(텍스트·데이터)가 주인공, 장식 최소.
- **컬러**: 중립 그레이 스케일 + 인디고 단일 액센트 유지. 카테고리별 컬러 남발 금지(중립 인식과 충돌).
- **시각적 금지**: 근육질·왁싱 바디 스톡이미지(그루밍≠몸매), 형광 네온, 의료 전후사진 썸네일 자극 사용.
- **WCAG 2.2 AA (§6.20)**: 본문 4.5:1 · 큰 텍스트/UI 3:1. 실측: 액센트 `#4f46e5`/흰배경 ≈ 6.3:1 ✓ · `--text-3 #6b7280` ≈ 4.8:1 ✓ · **`--text-4 #9ca3af` ≈ 2.5:1 — 본문 사용 금지, 장식·비필수 메타에만 허용** (규칙화). [미확정: 다크 테마 전 상태쌍 감사 — GRM-001 Lighthouse 감사와 병행]
- **DBA 등록부 (§6.15)** — 신생 브랜드라 전 자산이 "육성(emerger)" 단계:

| 자산 | Fame 현재 | Uniqueness | 4분면 | 역할 | 로고 없이 성립? |
|---|---|---|---|---|---|
| 워드마크 GROOMAN | ~0 (신규) | 중 (조어) | 육성 | 육성 — 유일 식별자 | — (자체가 로고) |
| 인디고 액센트 | 0 | 낮음 (범용색) | 육성 | 보조 — 단독 식별 기대 금지 | N |
| 심볼/캐릭터 | 없음 | — | — | [미확정 — 필요성 자체를 P2에서 재검토, 조기 투자 금지] | — |

- **모션·소닉 (§6.21)**: 모션 = 구현된 fade/slide 절제 유지(0.2~0.3s ease-out을 표준 이징으로 등재). 소닉 = 없음, 도입 계획 없음(웹 커뮤니티 — 소리는 오히려 반브랜드).

## 10. 브랜드 경험 원칙 (§19.5 역상황 행동 규약)

정서 선언이 아니라 **반대 상황의 행동 룰**:

| 상황 (스트레스 테스트) | grooman의 행동 규약 |
|---|---|
| 자동수집 글이 오분류·저품질로 노출됐다 | 숨기지 않는다 — 삭제·수정 후 처리 로그를 남긴다. 자동수집임은 항상 뱃지로 선표기 |
| 특정 병원·제품이 협찬을 제안한다 | 콘텐츠 상단 고정·후기 위장 ✕. 광고는 광고 지면에서 광고임을 표기하고 게재(§7 파트너 메시지) |
| 사용자가 의료적 판단을 요구한다("이거 탈모인가요?") | 진단하지 않는다 — 경험 공유 + "전문의 상담" 안내를 상수로 |
| 잘못된 정보가 HOT에 올랐다 | 인기여도 정정한다 — 정정 코멘트·출처 병기, 랭킹 조작은 하지 않는다 |
| 트래픽을 위해 자극 썸네일·어그로 제목이 유혹한다 | 제목은 내용을 초과하지 않는다(§8 정량화 톤) |

- 콘텐츠 경험: 모든 글에서 "이게 광고인지" 3초 안에 판별 가능해야 한다(뱃지·표기 체계).
- 운영/응대 기준의 상세 절차는 12 운영기획서 원본(경계 §12.3).
