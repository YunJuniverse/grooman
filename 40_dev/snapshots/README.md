# Snapshots

> 날짜별 산출물·리뷰·플랜·런북. 라이브 문서가 아니다.
> 파일명: `<type>-YYYY-MM-DD.md` (예: `business-plan-2026-05-03.md`)
>
> 자주 쓰이는 스냅샷 종류:
> - business-plan-YYYY-MM-DD.md (10번 기반)
> - architecture-YYYY-MM-DD.md
> - api-spec-YYYY-MM-DD.md
> - feedback-YYYY-MM-DD.md (환류 기록)
> - dev-master-plan-YYYY-MM-DD.md (마스터플랜 메이저 갱신 시점)
>
> 프롬프트: `50_resources/prompts/`
