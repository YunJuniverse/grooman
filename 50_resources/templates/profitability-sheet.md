## ③ 비용

| 비용 구분 | 산식 | 금액 |
|------|------|------|
| 직접경비 | — | |
| 외주비 | — | |
| 간접비 | 직접인건비 × 30% | |
| **리스크 컨틴전시** | 직접비 × 5~15% (근거=과거 초과율) | |

> 컨틴전시(known-unknowns, 베이스라인 내) ≠ 경영 리저브(unknown-unknowns, 베이스라인 밖·PMI). 임의 %가 아니라 *과거 초과 평균*을 factor로.

## ④ 수익현황

> **매출총이익(gross) = 수입 − 직접비(인건비+직접경비+외주)** / **영업이익(net) = gross − 간접비**. 두 마진을 분리해 본다.

| 항목 | 금액 |
|------|------|
| 수입 — 착수금 / 중도금 / 잔금 | |
| **수입 — 변경요청(change-order)** | (추가 스코프 매출 별도 집계) |
| 수입 합계 | |
| 지출 — 직접(인건비·경비·외주) | |
| 지출 — 간접 | |
| 지출 — 컨틴전시 | |
| 지출 합계 | |
| **매출총이익률(gross)% (=(수입−직접비)÷수입)** | |
| **영업이익률(net)% (=(수입−직접−간접)÷수입)** | |
| **손익분기(break-even) M/M 또는 매출** | |
| 누적 수익 / 목표 성취율 | |

**원칙**: 목표 마진(net) 15%+ 사수(참고 벤치마크: 커스텀 SW gross 40~60%). ① 간접비(=직접인건비×30%)를 invisible cost로 빠뜨리지 말 것 — 빠지면 흑자도 장부상 적자. ② **gross와 net을 분리**해 본다(하나의 blended 수익율은 어디서 새는지 못 봄). ③ 변경요청 매출·원가는 *별도 집계*(원 계약과 섞으면 마진 왜곡).
